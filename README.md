# OTT-share-service
팀프로젝트 - OTT 공유 플랫폼 서비스입니다.

- 회원 관리 기능
- 4명까지 가입할 수 있는 OTT 그룹 관리 기능
- 꿀잼 추천 게시판을 통해 이달의 추천작 및 추천글 관리 기능
